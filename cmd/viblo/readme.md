# Viblo bot

## Usage
```shell
./bot viblo -h
viblo editor choice / trending

Usage:
  discord viblo [flags]

Flags:
  -c, --channel string   channel id
  -h, --help             help for viblo
  -l, --limit int        Number of post (default 10)
  -o, --option string    EC for editor choices, T for trending (default "EC")
  -t, --token string     bot token

```
